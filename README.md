# Eleciton Analysis in an US Congressional Precinct in Colorado 
##Overview of Project 
This project demonstrates how Python can be used to create an automated process to calculate the popular vote in elections. 

###Purpose
The election commission is looking to generate code that provides a vote count report for each candidate in each county in the precinct.
